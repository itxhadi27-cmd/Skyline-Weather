# Skyline Weather

Skyline Weather is a sleek multi-city weather dashboard built to provide clean and simple weather insights.

## Live Demo
[View Live Project](https://itxhadi27-cmd.github.io/Skyline-Weather/)

## Overview
Skyline Weather allows users to search multiple cities and view weather information in a modern and easy-to-use interface.

## Features
- Multi-city weather search
- Current weather details
- Forecast display
- Location search
- Geocoding support
- Clean responsive layout
- API-driven interface

## Tech Stack
- React
- TypeScript
- HTML
- CSS
- Open-Meteo API
- OpenStreetMap Geocoding

## Purpose
This project shows that I can work with APIs, create elegant dashboards, and build polished user interfaces.
